A creative portfolio / shop, built upon WordPress.

## Features

- Dark mode switch

## Requirements

node v11.6.0 (npm v6.9.0).

Run 'npm install' from within the wp-content/themes/gulp-dev directory, then run 'gulp' for browser-sync and asset compilation.
